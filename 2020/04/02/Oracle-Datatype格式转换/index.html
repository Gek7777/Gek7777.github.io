Oracle中 Datatype格式转换

发表于 2020-04-02 更新于 2021-12-10 分类于 学习 Changyan： 本文字数： 778 阅读时长 ≈ 1 分钟
Oracle SQL中的数据类型转换，分为隐式转换和显示转换两种...

一、隐式数据类型转换

FROM TO
VARCHAR2 or CHAR NUMBER
VARCHAR2 or CHAR DATE
NUMBER VARCHAR2
DATE VARCHAR2

二、显式数据类型转换

FROM TO FUNCTION
CHARACTER NUMBER TO_NUMBER(VARCHAR2/CHAR)
CHARACTER DATE TO_DATE(VARCHAR2/CHAR)
NUMBER CHARACTER TO_CHAR(NUMBER)
DATE CHARACTER TO_CHAR(DATE)
