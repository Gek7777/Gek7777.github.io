Mysql-安装指引

发表于 2020-04-02 更新于 2022-10-21 分类于 学习 本文字数： 636 阅读时长 ≈ 1 分钟
Mysql的本地测试demo的建议安装教程

下载Mysql

国内链接：http://mirrors.sohu.com/mysql/MySQL-8.0/

下载后解压，放到想要放的位置即可。

配置环境变量

变量名：MYSQL_HOME

变量值：D:\Program Files\mysql-8.0.11-winx64

生成date文件

管理员身份运行-CMD

进入Mysql的bin目录：D:\Program Files\mysql-8.0.11-winx64\bin

执行命令： mysqld –initialize-insecure –user=mysql

在D:\Program Files\mysql-8.0.11-winx64\bin目录下生成data目录

安装Mysql

继续执行命令： mysqld -install

启动服务

继续执行命令： net start MySQL
